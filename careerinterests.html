Career Interests

front-end web developer

A front-end web developer develops functional applications using a variety of different coding languages, create mockups of websites, and help back-end developers with troubleshooting. Different companies have different requisites to become a front-end developer. Most require a lot of in-depth knowledge on how multiple different markup languages work. The median salary is $81,750.

full-stack developer

A full-stack developer is a web developer that has the skills to work with both ends of a website. They have to be familiar with multiple different markup languages, CSS, JavaScript, and a lot of back end languages. Most full-stack developers specialize in one section of development. The average salary for a full-stack developer is $95,628.
